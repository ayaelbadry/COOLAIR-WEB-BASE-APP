# Cool Air AC Web System

A custom web application developed for an air conditioning and water filtration company using Native PHP and MySQL.

The system streamlines customer interactions, order management, service bookings, and business operations through a centralized administrative dashboard.

---

## Features

- User Authentication & Authorization
- Role-Based Access Control (RBAC)
- Product Management
- Shopping Cart & Order Management
- Installation & Maintenance Booking
- AI Customer Support (Google Gemini API)
- Real-Time Live Chat (Pusher)
- Customer Reviews & Testimonials
- Multi-language Support (Arabic & English)
- Admin Dashboard
- Password Reset with OTP Verification
- Responsive User Interface

## Project Architecture

The application follows the MVC (Model–View–Controller) architecture and includes:

- Authentication Module
- Customer Module
- Product Module
- Booking Module
- Shopping Cart
- Order Management
- AI Chat Module
- Administration Dashboard
- Role & Permission Management
